Час виконання алгоритму зростає пропорційно кількості вершин та ребер у мережі, що підтверджує його практичну ефективність для даної моделі.**

**Завдання 2. Розширення функціоналу префіксного дерева**\
Реалізуйте два додаткових методи для класу Trie:\
count_words_with_suffix(pattern) для підрахунку кількості слів, що закінчуються заданим шаблоном;
has_prefix(prefix) для перевірки наявності слів із заданим префіксом.

**Висновок: алгоритм працює коректно, обчислює кількість слів, що закінчуються на заданий суфікс, та перевіряє наявність префіксів у наборі слів, дозволяючи оцінити пошукові можливості структури Trie. Результати показують, які слова містять певні суфікси, які префікси присутні, а також демонструють коректну обробку некоректних даних.**\
**Тестування на великих наборах слів (10 000+) підтверджує ефективність алгоритму: методи count_words_with_suffix та has_prefix працюють швидко і надійно, не залежно від обсягу даних. Час виконання зростає лінійно пропорційно довжині слів і кількості доданих елементів, що підтверджує практичну ефективність алгоритму для великих текстових наборів.**
